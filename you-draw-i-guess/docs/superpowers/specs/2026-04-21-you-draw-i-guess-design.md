# 你画我猜 - 游戏设计规格

## 概述

一个实时多人在线你画我猜游戏，使用 React + Node.js + Socket.io 构建。

## 技术架构

- **前端**：React 18 + Socket.io-client + Canvas API
- **后端**：Node.js + Express + Socket.io
- **通信**：WebSocket (Socket.io)

## 用户身份

### 用户ID生成
- 基于浏览器特征（userAgent + screen + timezone + language）生成唯一Hash
- 格式：8位十六进制字符串

### 显示名称
- 自动生成趣味名称，格式：`{形容词}的{动物}`
- 形容词库：神秘的、快乐的、勇敢的、温柔的、聪明的、可爱的...
- 动物库：猫、狗、兔子、狐狸、鹰、鲸鱼、熊猫、狮子...
- 每次进入随机生成

### 用户头像
- 基于用户ID使用Canvas绘制几何抽象图案
- 包含：圆形、三角形、方块组成的独特图案
- 颜色基于ID生成，确保同一用户生成相同头像

## 游戏流程

### 等待阶段
- 用户进入页面后自动加入房间
- 等待连接用户达到2人
- 显示当前在线人数

### 游戏开始
- 当用户数 >= 2时，随机选择一名绘画者
- 绘画者看到词组（其他人不可见）
- 绘画时间：2分钟
- 绘画者可随时点击"完成绘画"提前结束

### 猜题阶段
- 从绘画者完成或2分钟到时开始
- 其他用户在输入框输入猜测
- 1分钟倒计时
- 猜对者 +1分，游戏进入下一轮
- 若无人猜对，直接进入下一轮

### 轮次规则
- 每次轮换绘画者
- 每人每轮最多得1分（猜对）
- 绘画者本轮不得分

### 游戏结束
- 所有用户完成一轮后，展示最终得分排名
- 30秒倒计时后开始新一轮游戏
- 游戏持续进行

### 异常情况
- 用户数 < 2时游戏暂停，显示等待提示
- 新用户可随时加入，中断当前轮次并重新分配

## 绘画功能

### 画布
- 尺寸：800x600（桌面），响应式缩放（移动端）
- 白色背景
- 支持触摸和鼠标绘制

### 工具栏
| 工具 | 功能 |
|------|------|
| 画笔 | 默认黑色，可选颜色 |
| 橡皮擦 | 擦除绘制内容 |
| 颜色选择 | 12种颜色：黑、白、红、橙、黄、绿、青、蓝、紫、粉、棕、灰 |
| 撤销 | 回退上一次笔触 |
| 重做 | 恢复已撤销的笔触 |
| 清空 | 清除整个画布 |

### 笔触数据格式
```typescript
interface Stroke {
  tool: 'brush' | 'eraser';
  color: string;
  size: number;
  points: Array<{x: number, y: number}>;
}
```

## 视觉风格

### 主题
- 深色背景：#1a1a2e（深紫黑色）
- 卡片背景：#16213e（深蓝）
- 霓虹点缀：#0ff（青色）、#f0f（品红）、#ff0（黄色）

### 字体
- 标题：粗体，现代感
- 正文：清晰易读

### 动画
- 按钮hover：发光效果
- 用户加入/离开：淡入淡出
- 倒计时：脉冲动画

## 词组词库

本地词库包含约100个常用词汇，分三类：
- 动物：猫、狗、鱼、鸟、熊、兔子...
- 食物：苹果、面包、披萨、寿司...
- 物品：汽车、房子、手机、书...

## Socket.io 事件

### 客户端 → 服务器
| 事件 | 数据 | 说明 |
|------|------|------|
| join | {userId, name, avatar} | 用户加入 |
| draw | {stroke: Stroke} | 发送笔触 |
| clear | - | 清空画布 |
| guess | {text: string} | 提交猜测 |
| finishDrawing | - | 绘画者完成 |

### 服务器 → 客户端
| 事件 | 数据 | 说明 |
|------|------|------|
| usersUpdate | {users: User[]} | 用户列表更新 |
| gameStart | {drawer: userId, word: string} | 游戏开始（仅绘画者收到word） |
| drawerStatus | {drawer: userId} | 当前绘画者（非绘画者可见） |
| stroke | {stroke: Stroke} | 接收笔触 |
| clearCanvas | - | 清空画布 |
| guessResult | {correct: boolean, guesser: userId} | 猜测结果 |
| roundEnd | {winner: userId, scores: {}} | 本轮结束 |
| gameEnd | {rankings: []} | 游戏结束 |

## 里程碑

1. **项目初始化** - 创建React和Node项目，配置Socket.io连接
2. **基础页面结构** - 布局框架，用户列表，工具栏
3. **Canvas绘画功能** - 画笔、橡皮擦、颜色、撤销/重做
4. **实时同步** - 笔触数据实时传输和渲染
5. **游戏逻辑** - 用户加入、游戏开始、轮次切换、计分
6. **游戏结束流程** - 展示得分、重新开始倒计时
7. **移动端适配** - 响应式布局优化
